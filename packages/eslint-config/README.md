Personal ESLint configuration

## Getting Started

1. Add the package to your devDependencies
```bash
$ pnpm add -D eslint @ivanmaxlogiudice/eslint-config
```

2. Create a `.eslintrc` file
```bash
{
    "extends": ["@ivanmaxlogiudice/eslint-config"]
}
```
